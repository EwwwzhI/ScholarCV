## 科研经历

### 面向车载边缘计算的 3D 多模态感知算法研究 | 2023.09 - 2024.05 | 第一作者

- **研究背景：** 现有的 3D 多模态（视觉-雷达）融合感知算法由于计算量巨大，难以在车载边缘低功耗设备上实现高性能、高帧率的实时运行。
- **核心贡献：** 针对六路摄像头与毫米波雷达融合感知领域，独立提出一种面向边缘部署的实时轻量化视觉—雷达 BEV 感知框架 EdgeRCBEV。创新性地设计了高效的 RCS 感知雷达中间编码器与视觉流式历史帧推理机制。
- **成果指标：** 在权威数据集 nuScenes 上测试，于 NVIDIA Jetson AGX Orin 边缘平台上实现了 11.19 FPS 的实时帧率，较基准模型加速 16.2 倍，同时保持了 56.35% NDS 的极高感知精度。
- **科研成果：** 以第一作者身份撰写学术论文 《EdgeRCBEV: A Jetson-Based Real-Time and Lightweight Radar-Camera BEV Framework for 3D Object Detection》，目前在顶级期刊 IEEE TCSVT 在投。
- **负责工作：** 独立负责该课题的整体推进。完成多模态数据集的预处理与对齐；基于 PyTorch 编写核心网络与流式推理代码；使用 TensorRT 和 C++ 完成该复杂大模型在边缘计算板卡上的量化与交叉编译部署。
